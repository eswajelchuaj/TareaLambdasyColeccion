# Tarea de Lambdas y Clases de Coleccion

En este proyecto, se estarán realizando ejercicios/métodos con el enfoque principal siendo la familiarizacion con la estructura de las Lambdas, HashMaps y HashSets.

Este proyecto consiste de 10 métodos:
1. El Mulitplicador:
   El multiplicador consiste en un método que recibe un ArrayList de numeros enteros y los multiplica por el factor
   definido en el código.
2. El Filtro Selectivo:
   El filtro selectivo es un método que recibe un ArrayList de String, donde filtra y elimina las palabras que coincidan
   con la Inicial y/o el liminite minimo de letras.
3. Conversor de Mayúsculas:
   El conversor de mayúsculas recibe un ArrayList de String y convierte todo lo que esté en el ArrayList a mayúsculas
4. Cuadrados Únicos
   El método de cuadrados únicos recibe un ArrayList de tipo Integer, elimina los numeros duplicados y que no sean par, para convertirlos en numeros cuadrados
5. Mapa de Longitudes
   Este método recibe un ArrayList de Strings y crea un mapa donde cada palabra se convierte en una clave asociada a un número que representa su cantidad de letras.
6. Modificador de Inventario
   Es un método que recibe un HashMap con nombres de productos y sus precios, el cual utiliza un Bi-Consumer para imprimir un catálogo aplicando un descuento del 10% a cada artículo.
7. Contador de Frecuencia
   Este método recibe un ArrayList de palabras con repeticiones y construye un mapa que contabiliza cuántas veces aparece cada palabra individualmente usando la función merge.
8. Clasificador de Palabras
   El clasificador recibe un HashMap de frecuencias y un número mínimo; filtra el mapa para extraer y guardar en un ArrayList solo las palabras que se repitieron una cantidad de veces igual o mayor al límite.
9. Deduplicacion de Plabras
   Este método recibe una frase completa, la fragmenta en palabras individuales y filtra las que son muy cortas para convertirlas a minúsculas y asegurar que no haya repetidas usando un HashSet. 
10. El Tope de Frecuencias
   Es un método que recibe un HashMap de frecuencias y un valor máximo; revisa todos los valores del mapa y, si alguno supera el límite definido, lo recorta exactamente a ese número tope.
